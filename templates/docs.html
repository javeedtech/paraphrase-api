‚è±Ô∏è Rate Limits

To ensure fair usage and maintain service quality, the API implements rate limiting:

  • 60 requests per minute per IP address
  • Rate limits reset every minute
  • Exceeding limits returns HTTP 429 status code
Rate Limit Headers
HTTP/1.1 429 Too Many Requests
{
  "error": "Rate limit exceeded",
  "message": "Too many requests. Please try again later.",
  "retry_after": 60
}

‚ùå Error Codes

HTTP Code Error Type Description
400 Bad Request Invalid request format or missing required fields
429 Rate Limit Exceeded Too many requests within the rate limit window
500 Internal Server Error Server error during processing

import requests
import json

url = "https://your-api-domain.com/api/paraphrase"
headers = {"Content-Type": "application/json"}

data = {
    "text": "The quick brown fox jumps over the lazy dog",
    "max_length": 100,
    "temperature": 0.7
}

response = requests.post(url, headers=headers, data=json.dumps(data))
result = response.json()

print(f"Original: {result['original_text']}")
print(f"Paraphrased: {result['paraphrased_text']}")
const paraphraseText = async (text) => {
  const response = await fetch('https://your-api-domain.com/api/paraphrase', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      text: text,
      max_length: 100,
      temperature: 0.7
    })
  });

  const result = await response.json();
  console.log('Paraphrased:', result.paraphrased_text);
  return result;
};
